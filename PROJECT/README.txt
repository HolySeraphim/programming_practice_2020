Для старта:

1) Перейти по: https://vk.com/sovushka_bot
2) Нажать: "Добавить в беседу"
3) Запустить main.py
4) Запустить update_bot.py

Команды:
1) Старт (Start) - начать работу с ботом
2) Отправляй совушек - подписаться на рассылку сов в беседу
3) Не отправляй совушек - отписаться от рассылки сов в беседу
4) Забрать совушку - взять присланную в беседу сову
5) Моя совушка - получить информацию о своей сове
6) Переименовать TEXT - дать сове имя TEXT
7) Выбросить совушку - выбросить сову
8) Да, подтвердить удаление - подтвердить действие "выбросить сову"
9) Нет, хочу оставть совушку - отменить децствие "выбросить сову"
10) Покормить совушку - покормить сову
11) Совушка работа - отправить/(забрать) совушку на/(с) работы
12) Купить конфету - купить конфету
13) Съесть конфету - съесть конфету

Файлы:
update_bot - осуществляет рассылку сов по подписанным на рассылку беседам каждые 1,5 часа
main - запускает программу, проверяет рассылку и реагирует на команды от пользователя
config - содержит основные параметры связи бота с VK
API - содержит основные функции бота, связанные с vk_api, в том числе и подключение к серверу
keyboard - содержит все функции с параметрами клавиатуры для быстрого использования при написании кода
owl_database - содержит функции, вызываемые из main командами пользователя, связывается с pickle
Owl - задаёт параметры совы
